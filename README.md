# ArcticFlow-Analytics-

ArcticFlow Analytics is an intelligent fuel management dashboard built for Alaska's harsh winters. 
This repository contains the premium frontend implementation of the application.

## Features
- Glassmorphic wintery aesthetic design system (Dark mode)
- Real-time animated fuel tank dashboard monitors
- Integrated K-Factor calculator algorithm
- Chart.js 7-Day weather impact predictions
- Mobile-responsive navigation and data views
